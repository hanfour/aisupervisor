AI SUPERVISOR

VIRTUAL OFFICE FOR AI WORKERS

A macOS desktop app that manages AI workers running Claude Code CLI in tmux sessions. Assign tasks, track progress, and review code — all from a pixel-art office interface.

⬇ Download for macOS v1.0.0 — macOS 12+

FEATURES

🤖

AI WORKER MANAGEMENT

Hire AI workers with different skill profiles — coder, architect, designer, analyst. Each runs Claude Code CLI in an isolated tmux session with task-specific git branches.

📋

PROJECT PIPELINE

Create projects with goals, auto-decompose into tasks with AI, track on a kanban board. Full PRD pipeline with human gate approval before development starts.

🔍

CODE REVIEW SYSTEM

Automated review pipeline where reviewer workers check code quality. Approved work merges automatically; rejected tasks get reworked with feedback.

🏢

PIXEL OFFICE

Watch your AI team in a retro pixel-art office. Each worker has personality traits, moods, relationships, and customizable appearances. NPC dialogue included.

REQUIREMENTS

  • macOS 12 (Monterey) or later
  • Claude Code CLI installed (npm install -g @anthropic-ai/claude-code)
  • Anthropic API key configured in Claude CLI
  • tmux and git (bundled with the app, or install via Homebrew)
